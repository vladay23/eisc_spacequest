ℹ️ Принцип работы

При генерации документа система проверяет, есть ли активный шаблон для данного типа.

Если шаблон загружен — система заполняет плейсхолдеры {{'{{'}}КЛЮЧ{{'}}'}} реальными данными.

Если шаблон не загружен — документ генерируется автоматически с профессиональным оформлением.

Для каждого типа может быть активен только один шаблон. При загрузке нового шаблон того же типа деактивируется.
